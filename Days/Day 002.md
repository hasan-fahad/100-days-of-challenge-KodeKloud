## ğŸ’¡ Good to Know

### **User Account Expiry**
- **Purpose**: Automatically disable accounts after a set date  
- **Format**: `YYYY-MM-DD` (ISO 8601)  
- **Check Expiry**:  
  ```bash
  chage -l username
  ```
- **Extend Expiry**:  
  ```bash
  sudo chage -E 2024-12-31 username
  ```

### **Temporary Account Management**
- Always set an expiry date for temporary accounts  
- Use `chage -l` to monitor account status  
- Expired accounts remain but cannot log in  
- Remove accounts completely:
  ```bash
  sudo userdel -r username
  ```
